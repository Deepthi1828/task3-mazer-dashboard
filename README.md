# Task 3 – Mazer Admin Dashboard Customization
Task 3 project is built using the Mazer Bootstrap admin dashboard template.  This project focuses on customizing the UI and integrating sample data  to build a responsive and functional dashboard.

## Overview
This project is completed as part of **Task 3** of the internship assessment.  
The objective of this task is to work with an existing admin dashboard template, customize the UI, and bind components with data to create a functional and responsive dashboard.

The **Mazer Bootstrap Admin Dashboard** template is used as the base for this project.


## Task Requirements
As per the Task 3 instructions, the following requirements are fulfilled:

- Use the **Mazer Admin Dashboard** template as the base project  
- Customize the **UI/UX**, layout, and components
- Bind dashboard components with **sample JSON data or API data**
- Ensure the dashboard is **responsive and functional**
- Maintain clean project structure and readable code
- Host the complete project in a **GitHub repository**
- Provide proper documentation using a **README file**
